# EZ-WebView
This is a WKWebView based web view framework which supports for swift UI. 

Installation

pod 'EZ-WebView'

Available versions are -> 

pod 'EZ-WebView','~> 0.1.4'
pod 'EZ-WebView','~> 0.1.5'
pod 'EZ-WebView','~> 0.1.6'
pod 'EZ-WebView','~> 0.1.7'
